Январское обновление HavocOS

Январь 19, 2023

СПИСОК ИЗМЕНЕНИЙ:

• Обновлен патч безопасности до января 2023 года

• Восстановлены прошивки для recovery для устройств с рабочей версией TWRP.

• Экран выбора языка ввода давал сбой для некоторых клавиатур (например, OpenBoard)

Внимание для пользователей Xiaomi 12T Pro

Из-за отсутствия библиотек для основной камеры для данного устройства, камера не будет работать должным образом пока не будет выпущена глобальная прошивка на базе Android 13.
